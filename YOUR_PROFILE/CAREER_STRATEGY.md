## COMPENSATION SCREEN

Base-comp bands for the current market:
- floor: `180k` base
- target: `205k-225k` base
- stretch: `230k+` base

Use these as the default screen:
- if disclosed base comp is below `180k`, default to `pass`
- allow `watch` below `180k` only if the company/problem quality is unusually strong or the upside case is unusually credible
- `180k-204k` can be viable, but should usually need either high interest, strong scope, or unusually good upside
- `205k-225k` is the target band for senior PM roles that fit well
- `230k+` is premium and should be treated as strong comp support for a role
- if compensation is missing, do not auto-pass on comp alone
- do not let a role become `ready_to_apply` when the comp looks weak and the interest level is only medium

Startup exception:
- lower cash can be acceptable only when equity quality is plausibly strong and the role is genuinely high-interest
